¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje utilizado para diseñar y dar estilo a las páginas web.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite implementar funcionalidades interactivas en las páginas web.
